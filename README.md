# 📊 MarketPulse – Real-Time Stock Dashboard

An interactive stock market analytics dashboard built using Streamlit, Plotly, and Yahoo Finance API. The dashboard provides real-time market insights with live stock data visualization and performance analysis.

## 🚀 Features

- 📈 Top Gainers tracking
- 📉 Top Losers tracking
- 📊 Volume Leaders analysis
- 🔍 Search stock functionality
- 📋 Live market data updates
- 📈 Interactive charts and visualizations
- 📊 Market overview scatter analysis
- 🎨 Attractive dashboard UI with finance-style theme
- ⚡ Fast data loading using caching

---

## 🛠 Technologies Used

- Python
- Streamlit
- Plotly
- Pandas
- Yahoo Finance API (yfinance)

## 📥 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/NSE-Market-Dashboard.git
```

Move into project directory:

```bash
cd NSE-Market-Dashboard
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Run Application

Start the Streamlit server:

```bash
streamlit run stock_dashboard.py
```

Application opens in browser:

```bash
http://localhost:8501
```

## 📊 Dashboard Modules

### Top Gainers
Displays stocks with the highest percentage increase.

### Top Losers
Displays stocks with the highest percentage decrease.

### Volume Leaders
Shows most actively traded stocks.

### Market Overview
Interactive visualization of stock movement and market trends.

---

## 🎯 Project Objective

The objective of this project is to provide users with an interactive platform for monitoring live market activity and identifying important market trends such as gainers, losers, and high-volume stocks.
